Namig

Sprehodi se po elementih tabele in za vsakega preveri, ali je manjši od 0. Če je, ga izpiši.

Rešitev

# Program izpiše vse negativne elemente v podani tabeli.
tabela = [-6, 10, 7, -11, 0, 3, 10, -5]

for element in tabela:
  if element < 0:
    print(element)

Namig

Sprehodi se po elementih tabele in za vsakega preveri, ali je deljiv s 3. Če je, ga izpiši.

Rešitev

# Program izpiše vse večkratnike števila 3 v podani tabeli.
tabela = [18, 16, 9, 18, 31, 30, 16, 15]

for element in tabela:
  if element % 3 == 0:
    print(element)

Namig

Nastavi števec na 0, nato pa se sprehodi po elementih tabele in za vsakega preveri, ali je manjši od 0. Če je, povečaj števec za 1. Števec na koncu izpiši.

Rešitev

# Program izpiše število negativnih elementov v podani tabeli.
tabela = [-6, 10, 7, -11, 0, 3, 10, -5]
stevec = 0

for element in tabela:
  if element < 0:
    stevec += 1

print(stevec)

Namig

Sprehodi se po indeksih tabele in vsakokrat preveri, ali je element na trenutnem indeksu manjši od 0. Če je, izpiši trenutni indeks.

Rešitev

# Program izpiše indekse vseh negativnih elementov v podani tabeli.
tabela = [-6, 10, 7, -11, 0, 3, 10, -5]

for indeks in range(len(tabela)):
  if tabela[indeks] < 0:
    print(indeks)

Naloge

1

Napiši program, ki izpiše vse negativne elemente v podani tabeli. Tabela naj vsebuje samo cela števila. (To velja tudi za naslednje naloge.) Na primer, za tabelo 

tabela = [-6, 10, 7, -11, 0, 3, 10, -5]
naj program izpiše števila −6, −11 in −5.

2

Napiši program, ki izpiše vse večkratnike števila 3 v podani tabeli. Na primer, za tabelo 

tabela = [18, 16, 9, 18, 31, 30, 16, 15]
naj program po vrsti izpiše števila 18, 9, 18, 30 in 15.

3

Napiši program, ki izpiše število negativnih elementov v podani tabeli. Na primer, za tabelo 

tabela = [-6, 10, 7, -11, 0, 3, 10, -5]
naj program izpiše število 3.


4

Napiši program, ki izpiše vse indekse negativnih elementov v tabeli. Na primer, za tabelo 

tabela = [-6, 10, 7, -11, 0, 3, 10, -5]
naj program izpiše indekse 0, 3, 7.
